Support Maya version¶

The cymel Python version supports Maya 2015 and later.

The C++ version will support Maya 2012 and later.

Also works with Python 3 in Maya 2022.

install¶

Either clone or download the cymel GitHub repository and put your Python path in the python folder (it contains one folder called cymel), or copy the cymel folder to an existing Python path.

The cymel folder contains all the files needed to run cymel.

That’s all.
