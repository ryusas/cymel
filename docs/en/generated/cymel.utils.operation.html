cymel.utils.operation.docmd(do, undo, redo=None)¶

Given an arbitrary callable object, execute it as an undoable command.

Processes that would otherwise be impossible to undo/redo can be executed as undoable.

Parameters:

  • do – Executable object for first run.

  • undo – Executable object for undo.

  • redo – Executable object for redo. If omitted, do is used.

>>> import maya.cmds as cmds
>>> import cymel.main as cm
>>> v = cmds.jointDisplayScale(q=True)
>>> cm.docmd(lambda: cmds.jointDisplayScale(v*10), lambda: cmds.jointDisplayScale(v))
>>> cmds.undo()
>>> v == cmds.jointDisplayScale(q=True)
True
