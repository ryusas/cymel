cymel.qt.widgets.getWidgetByPathName(name, base=None, separator='|', includesLayout=True)¶

Find a widget or layout by path of objectName.

Unknown widgets that exist between specified paths are also allowed and searched thoroughly.

Parameters:

  • name (str) – Path name below the base point (not including the base point).

  • base – Base widget. If omitted, the first name in the path is searched from the top level. If it is still not found, it becomes the main window of the application obtained by getFirstTopLevelWindow.

  • separator (str) – Path delimiter.

  • includesLayout (bool) – Whether layout managers also count as nodes.

Return type:

QWidget or QLayout

cymel.qt.widgets.showAppWindow(proc, deleteOnClose=True, setupRoundingOnNewApp=False, traceSettings=False)¶

Generate and display a window on an existing or new QApplication.

If QApplication is uninitialized, it is initialized, enters the main loop, and does not return. In an embedded environment, it becomes a child of the main window (or top-level window) and is returned.

Parameters:

  • proc (callable) – An executable object, such as a widget type or function, for creating widgets.

  • deleteOnClose (bool) – Whether to be deleted when closed. WA_DeleteOnClose attribute is set to override.

  • setupRoundingOnNewApp (bool) – In a newly opened application, and if there is one or more screens with devicePixelRatio that is a non-integer in Qt6 or later, set HighDpiScaleFactorRoundingPolicy to RoundPreferFloor. This is to avoid artifacts when using anti-aliasing in QGraphicsView. By the way, in Maya, QT_ENABLE_HIGHDPI_SCALING=0 is set, so all devicePixelRatios are 1.0 and no problem occurs.

  • traceSettings (bool) – Print out the Qt version, application settings, etc.

Return type:

QWidget

cymel.qt.widgets.showWindow(wd, deleteOnClose=True)¶

Display the widget as a window.

It becomes a child of the application’s main window or an already visible top-level window (the first one), if one exists.

Parameters:

  • wd (QWidget) – widget.

  • deleteOnClose (bool) – Whether to be deleted when closed. WA_DeleteOnClose attribute is set to override.

cymel.qt.widgets.unscaleTextFont(textCtl, height=None, pixel=11, prepare=None)¶

Get the inverse scale value and QFont or None for fixed size text.

Parameters:

  • textCtl – A text control used to determine size. Must have boundingRect, font, and setFont methods.

  • height (float) – Specifies the reference height when the font is specified in point size. A scale value is calculated to match this size. The default None means a default value that changes depending on the environment.

  • pixel (int) – Specifies the standard size when the font is specified in pixel size. A scale value is calculated to match this size.

Paream callable preparecallable prepare:

If information cannot be obtained from the cache and preparation (temporary drawing, etc.) is required before checking the point size height, specify the callable object.

Return type:

float, QFont or None
