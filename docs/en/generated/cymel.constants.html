cymel.constants¶

Main constants used throughout cymel.

It may be a good idea to import it with * as it has been carefully selected for frequently used items.

Constants:

cymel.constants.AVOID_ZERO_DIV_PRECISION = 1e-13¶

In Maya, even if you set scale=0, the matrix value seems to be maintained at around 1.e-12 without being crushed, so keep this in mind when setting the minimum value.
