cymel.qt.binding¶

PySide binding determination and compatibility aliases.

Names derived from Qt/PySide are published as they are.

  • QtCore

  • QtGui

  • QtWidgets

  • Qt

  • Property

  • Signal

  • Slot

  • QWidget

  • QApplication

Binding information added by cymel.

  • QT_BINDING_NAME (str): Binding name in use.

  • QT_BINDING_VERSION (str): Binding version string.

  • QT_BINDING_VERSION_I (tuple): Numeric tuple of binding version.

  • QT_VERSION (str): Qt version string.

  • QT_VERSION_I (tuple): A numerical tuple of Qt version.

  • QT_MAJOR_VERSION (int): Major version of Qt.

  • qtmodule: PySide module in use.

  • uic: UI compiler module for PySide/PySide2. None if there is none.

Compatibility helper from shiboken.

  • wrapInstance: Convert a C++ pointer to a Qt wrapper.

  • unwrapInstance: Get C++ pointer from Qt wrapper.

  • isValidWrap: Returns whether the Qt wrapper is valid.

  • isInvalidWrap: Returns whether the Qt wrapper is invalid.

  • isOwnedByPython: Returns whether the instance is owned by Python.

  • deleteInstance: Delete the target instance of Qt wrapper.
