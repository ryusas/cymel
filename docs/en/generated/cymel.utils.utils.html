cymel.utils.utils.createNode(nodetype, **kwargs)¶

Generate a node.

Acts as a replacement for the createNode command.

CyObject is returned while the node type name is replaced by compat_nodetype for compatibility between old and new versions.

cymel.utils.utils.enablePlugin(name, proc=<function loadPlugin>)¶

Make the plugin available.

Use this when you do not want an error to occur even if the plugin does not exist.

Returns a bool value indicating success or failure.

By default, loadPlugin is called, so if the plugin is not loaded, it will be loaded, but if the plugin does not exist, no error will occur and retries will be suppressed when the same name plugin is called for the second time or later.

Parameters:

  • name (str) – Plugin name.

  • proc (callable) – Function used to load plugins. By default, loadPlugin is used.

Return type:

bool

cymel.utils.utils.listEnum(name, attrname=None, reverse=False)¶

Get a list of enum attribute name/value pairs.

The specification method differs depending on whether it is a static attribute or a dynamic attribute.

Parameters:

  • name (str) – Plug name for dynamic attributes, node type for static attributes.

  • attrname (str) – Attribute name for static attributes.

  • reverse (bool) – Reorder the pairs so that they are (value, name) instead of (name, value).

Return type:

list
