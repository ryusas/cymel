cymel.utils.files.exportSceneFile(fname, sel=None, all=False, ftype=None, **kwargs)¶

Export the scene file.

Parameters:

  • fname (str) – Scene file name. If the directory does not exist, it will be automatically created.

  • sel – Names or list of nodes to output when all=False. If omitted, the selected item will be exported.

  • all (bool) – Whether to export the entire scene.

  • ftype (str) – File type name. Can be omitted if it can be inferred from the extension.

Return type:

str

cymel.utils.files.importSceneFile(fname, namespace=None, prefix=None, **kwargs)¶

Import the scene file.

Parameters:

  • fname (str) – Scene file name.

  • namespace (str) – Specify this when adding a namespace to the imported node. If you specify a true value that is not a string, it will be automatically determined from the file name.

  • prefix (str) – Specify this when adding a prefix to the imported node. If you specify a true value that is not a string, it will be automatically determined from the file name.

Return type:

str

Save the scene file.
