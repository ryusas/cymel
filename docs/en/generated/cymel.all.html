cymel.all¶

Recommended process for importing full functionality.

You can easily import all functions by doing the following:

from cymel.all import *

This will do the following:

from .constants import *
from . import main as cm
from . import ui as cmu

import maya.cmds as cmds
import maya.mel as mel
import maya.api.OpenMaya as api
import maya.api.OpenMayaAnim as apia

import maya.utils
import pprint
maya.utils.formatGuiResult = pprint.pformat
