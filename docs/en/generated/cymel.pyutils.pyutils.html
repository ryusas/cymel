cymel.pyutils.pyutils.avoidZeroDiv(v, pre=1e-13)¶

Obtain a denominator to prevent zero division by imitating Maya.

In Maya, even if scale=0 is set, 0.0 is limited and held at around 1.e-12 so that matrix does not become a singular value, so the value multiplied by the limit with this in mind is returned.

Parameters:

  • v – Value to check.

  • pre – Minimum value. Default is AVOID_ZERO_DIV_PRECISION.

Return type:

リミットを適用した値。

cymel.pyutils.pyutils.insertEnvPath(path, name, index=-1, noCheck=False, noUpdate=False)¶

Add paths to various path environment variables.

Returns 0 if the path does not exist and was not added, 1 if it is already in the list, and 2 if it is newly added.

Parameters:

  • path (str) – Path to add.

  • name (str) – Environment variable name.

  • index (int) – Position to add. If it is a negative number, it will be added to the end, otherwise it will be inserted before this position.

  • noCheck (bool) – Do not check whether the path exists. By default, paths that do not actually exist are not added.

  • noUpdate (bool) – Do not update if path is already included. By default, if a path is already in the list, it will be removed and re-added.

Return type:

int

cymel.pyutils.pyutils.insertSysPath(path, index=-1, noCheck=False)¶

Add path to sys.path.
