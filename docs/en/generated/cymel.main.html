cymel.main¶

All functions except UI.

All of the following are expanded in parallel and can be easily accessed.

  • sel or selection representing the current selection.

  • Everything below cymel.core (major node classes, data types, etc.).

  • nt (another name for an instance of NodeTypes) that has access to all node classes.

  • Node type name map cntm (another name for instances of CompatNodeTypeMap) for compatibility with new and old versions of Maya.

  • All about cymel.constants.

  • Everything below cymel.pyutils.

  • Everything under cymel.utils.

  • Everything about cymel.initmaya.
