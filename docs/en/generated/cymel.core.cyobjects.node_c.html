cymel.core.cyobjects.node_c.keyForBreadthFirst(node)¶

Key function for breadth-first sorting of the node list in the DAG hierarchy.

If it is a DAG node, it returns the result of lengthAndSiblingIndices, otherwise it returns 0 and an empty list.

If you don’t mind that the order of nodes with the same path length is not guaranteed, using keyForPathLength will make processing a little easier.

Parameters:

node (Node) – Node to inspect.

Ryype:

(int, list)

cymel.core.cyobjects.node_c.keyForDepthFirst(node)¶

Key function for sorting the node list by DAG hierarchy depth first.

Returns the result of siblingIndices if it is a DAG node, otherwise returns an empty list.

cymel.core.cyobjects.node_c.keyForPathLength(node)¶

Key function to sort the node list by DAG path length (hierarchy depth).

Similar to keyForBreadthFirst, but simpler and does not guarantee the ordering of nodes with the same path length.

Returns the result of pathLength if it is a DAG node, otherwise returns 0.

Parameters:

node (Node) – Node to inspect.

Return type:

int
