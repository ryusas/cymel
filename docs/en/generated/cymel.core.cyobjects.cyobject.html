Function Details:

cymel.core.cyobjects.cyobject.cyObjects(val)¶

Get a list of CyObject from name list etc.

It can accept either a string or a list of strings, and a value like None is interpreted as an empty list, so the return value of a Maya command can often be accepted as is.

No matter what the arguments are, the return type will be list.

Parameters:

val (iterable) – Node name, plug name, equivalent, or a list of them.

Return type:

list
