class cymel.core.cyobjects.cyobject.CyObject¶

class cymel.core.cyobjects.cyobject.CyObject(src)¶

Bases: object

Abstract base class for Maya wrapper objects.

Wrapper classes are broadly divided into Node and Plug.

By specifying a name to identify an existing node or plug in the scene or a Python API 2.0 object to the CyObject constructor, you can obtain an instance of the appropriate class.

Node classes are prepared or automatically generated for each node type. The system provides only one type of plug class. You can create custom classes by inheriting any of these classes.

For node classes, it is convenient to register them in the system with nodetypes.registerNodeClass.

For plug classes, it is convenient to specify the class to be used temporarily using CyObject.setGlobalPlugClass or Node.setPlugClass.

classmethod byUUID(val, candidates=(), py=False)¶

Get node list from UUID.

The return value is a list since duplicates are possible.

Parameters:

  • val (UUID) – A single UUID or a list of them. The type of each UUID can be specified as UUID or str.

  • candidates (iterable) – Specify this when you want to narrow down the candidates in advance when py=True.

  • py (bool) – Obtain it from what is stored in the attribute called uuid instead of Maya standard functionality. For versions below 2016, it is always treated as True.

Return type:

list

classmethod checktype(obj)¶

Checks whether the specified object matches the class.

If a CyObject derived object is specified, it will be returned if it matches, or None if it does not match.

If you specify any other value (such as a string or a Python API 2.0 object that can be specified to the CyObject constructor), a CyObject derived object of the appropriate type is returned if there is a match, otherwise None is returned.

Parameters:

obj – Object to check.

Return type:

CyObject or None

internalData()¶

Return internal data.

Override to extend internal data in a derived class. In that case, also override the newObject class method to support extension.

The internal data is assumed to be a black box, and the extended data must also include the base data.

isAlive()¶
