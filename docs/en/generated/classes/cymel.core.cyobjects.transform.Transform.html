class cymel.core.cyobjects.transform.Transform¶

class cymel.core.cyobjects.transform.Transform(*args, **kwargs)¶

Bases: TransformMixin, DagNode

transform Node type wrapper class.

Supports node generation when creating a class instance without fixed arguments.

Methods:

addChild(child[, r, add, avoidJointShear, ...])

Add child nodes. Multiple specifications are possible.

Attributes:

TYPE_BITS = 3¶

Represents the characteristics of nodes supported by the class.

Method Details:

addChild(child, r=False, add=False, avoidJointShear=False, unmaintainIS=False)¶

Add child nodes. Multiple specifications are possible.

Parameters:

  • child – DagNode or names of child nodes, or a list of them.

  • r (bool) – Whether to keep the current local transformation. By default it is maintained in world space.

  • add (bool) – Add path instead of move (instance).

  • avoidJointShear (bool) – If the child to be added is a joint and shear is required to maintain the world attitude, a transform will be added to avoid using it. This is the original behavior of Maya, but the default of this method is to use the shear of the joint and not add a transform.

  • unmaintainIS (bool) – If this node is a joint, inverseScale connection/disconnection is not maintained.
