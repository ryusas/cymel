setM(m, ws=False, safe=False, get=False)¶

Alternative name for setMatrix.

setMatrix(m, ws=False, safe=False, get=False)¶

Set the matrix.

If ws=True, it matches the result of performing setQuaternion’s ra=True, setScaling and setShearing, and setTranslation’s at=4 in order (the order of Scaling and Shear does not matter, but the others must be done in this order).

If ws=False, if segment scale correction of joint is effective, nothing other than setTranslation will match.

Parameters:

  • m (Matrix) – Matrix you want to set.

  • ws (bool) – Whether to set it in world space.

  • safe (bool) – An error does not occur even if the attribute cannot be set because it is locked. Also, only the places where double3 can be set are set.

  • get (bool) – It doesn’t actually set it, only calculates the attribute value that should be set.

Return type:

Transformation or None

setQ(q, ws=False, ra=False, r=True, jo=True, safe=False, get=False)¶

Alias ​​for setQuaternion.

setQuaternion(q, ws=False, ra=False, r=True, jo=True, safe=False, get=False)¶

Sets the node rotation quaternion.

By default, the rotation does not include rotateAxis, so it does not match the decomposition result from the matrix, but it matches the criteria for the rotation direction of the node by Maya’s functions (Local Axis display and constraints).

If ws=False, it means the value is composited from only the three rotation attributes, and the influence of inverseScale included in the joint node’s local matrix is ​​ignored.

The attribute to set is usually rotate, but if r=False it will be jointOrient or rotateAxis depending on the other options.

Parameters:

  • q (Quaternion) – The quaternion you want to set.

  • ws (bool) – Whether to set it in world space.

  • ra (bool) – Does it contain rotateAxis?

  • r (bool) – Does it contain rotate?

  • jo (bool) – Does it contain jointOrient?

  • safe (bool) – An error does not occur even if the attribute cannot be set because it is locked. Also, only the places where double3 can be set are set.

  • get (bool) – Does not actually set it, only calculates the rotate attribute value that should be set.

Return type:

list or None

Warning

Combinations of the following options will result in an error.

  • ws=False, ra=True, r=False, jo=True

  • ws=True, jo=False

  • ws=True, ra=True, r=False

  • ws=True, r=False and not joint

setScaling(v, ws=False, safe=False, get=False)¶

Sets the node’s scaling value.

If ws=False, it simply means the value of the scale attribute, and the influence of inverseScale contained in the joint node’s local matrix is ​​ignored.

Parameters:

  • v (sequence) – The scale value you want to set.

  • ws (bool) – Whether to set it in world space.

  • safe (bool) – An error does not occur even if the attribute cannot be set because it is locked. Also, only the places where double3 can be set are set.

  • get (bool) – Gets the scale attribute value that should be set without actually setting it.

Return type:

list or None

Warning

If non-uniform scaling or shear transformation that has not been corrected for segment scale exists at the top, it will also be affected by the rotation of this node, so the rotation must be set first.

setShearing(v, ws=False, safe=False, get=False)¶

Sets the node shear value.

If ws=False, it simply means the value of the shear attribute, ignoring any inverseScale effects contained in the joint node’s local matrix.

  • ws (bool) – Whether to set it in world space.

  • safe (bool) – An error does not occur even if the attribute cannot be set because it is locked. Also, only the places where double3 can be set are set.

  • get (bool) – Does not actually set it, only calculates the shear attribute value that should be set.

Return type:

list or None

setTransformation(x, ws=False, safe=False, get=False)¶

Set the transformation.

Modifier attributes such as pivot and jointOrient are also set.

When copying the transformations of the joint and transform nodes, if there are mutually incompatible attributes (pivot, segmentScaleCompensate, jointOrient, etc.), the matrices are set to match.

joint nodes treat pivot attributes as unsupported, but shear uses them.

Parameters:

  • x (Transformation) – Transformation information you want to set.

  • ws (bool) – Whether to set it in world space.

  • safe (bool) – An error does not occur even if the attribute cannot be set because it is locked. Also, only the places where double3 can be set are set.

  • get (bool) – It doesn’t actually set it, only calculates the attribute value that should be set.

Return type:

Transformation or None

setTranslation(v, ws=False, at=2, safe=False, get=False)¶

Set the position of the node.

By default, it means the position of the rotation pivot, which does not match the decomposition result from the matrix, but matches the criteria for the node’s position by Maya’s functions (Local Axis display position and constraints).

Parameters:

  • v (sequence) – The position you want to set.

  • ws (bool) – Whether to set it in world space.

  • at (int) – Which attribute should be set at the corresponding position? - If 0, local origin (parent matrix position). - If it is 1, it is the position of translate. - If 2, rotatePivot position. - 3 is the scalePivot position. - If it is 4 or more, it is the matrix position.

  • safe (bool) – An error does not occur even if the attribute cannot be set because it is locked. Also, only the places where double3 can be set are set.

  • get (bool) – Does not actually set it, only calculates the shear attribute value that should be set.

Return type:

list or None

Warning

If at=4, it must be called after setScaling, setShearing, and setQuaternion. Otherwise, they will change the position if the pivot is not at the local origin.
