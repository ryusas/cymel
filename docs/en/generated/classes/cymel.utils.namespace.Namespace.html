class cymel.utils.namespace.Namespace¶

class cymel.utils.namespace.Namespace(ns='')¶

Bases: str

Maya namespace class.

It can also be used as a context to control the current namespace.

Since it is a derived class of string, it behaves like a normal string.

An instance can be created whether or not it actually exists in Maya. If it does not exist, it will be created when making it current.

Immutable strings are always treated as absolute names starting with :, regardless of Maya’s relative namespace mode.

Warning

When setting the current namespace, be careful about the difference in behavior depending on Maya’s relative namespace mode setting.

  • Node generation will be named in the current namespace regardless of the relative namespace mode setting.

  • Node name evaluation and specification are not affected by current namespace specification unless the relative namespace mode setting is enabled.

Namespace handling in this class is not node generation, so relative names function effectively only when relative namespace mode is enabled.
