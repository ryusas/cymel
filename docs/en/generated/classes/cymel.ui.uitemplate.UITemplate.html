class cymel.ui.uitemplate.UITemplate¶

class cymel.ui.uitemplate.UITemplate(name=None, force=False, **kwargs)¶

Bases: object

uiTemplate wrapper class for mel UI.

setUITemplate can be done with with.

import maya.cmds as cmds
import cymel.ui as cmu

template = cmu.UITemplate('ExampleTemplate', force=True)
template.define(cmds.button, width=100, height=40, align='left')
template.define(cmds.frameLayout, borderVisible=True, labelVisible=False)

wnd = cmu.Window()
with template:
    cmu.FrameLayout()
    cmu.Button()
wnd.show()

Initialization.

Parameters:

  • name (str) – UI template name. If omitted or a name that does not exist is specified, a new name will be generated.

  • force (bool) – If an existing UI template name is specified, whether to delete the existing one and recreate it.
