class cymel.pyutils.pyutils.Singleton¶

class cymel.pyutils.pyutils.Singleton(clsname, bases, attrs)¶

Bases: type

Metaclass for creating singleton classes.

Only one instance of that type will be created.

When an instance is created, if an instance of the same type already exists, it will only be returned. Please note that it is used without taking into account the difference in argument contents at the time of generation.

>>> import cymel.main as cm
>>> class Foo(cm.with_metaclass(cm.Singleton, object)):
...     def __init__(self, v):
...         self._v = v
...     def __repr__(self):
...         return '%s(%s)' % (self.__class__.__name__, repr(self._v))
...
>>> class Bar(Foo):
...     pass
...
>>> a,b,c,d = Foo(1), Foo(2), Bar(3), Bar(4)
>>> a,b,c,d
(Foo(1), Foo(1), Bar(3), Bar(3))
>>> a is b
True
>>> a is c
False
>>> c is d
True
>>> del a,b,c,d
>>> b = Foo(2)
>>> b
Foo(2)
>>> del b
