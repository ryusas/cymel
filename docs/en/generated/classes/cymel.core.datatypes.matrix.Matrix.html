class cymel.core.datatypes.matrix.Matrix¶

class cymel.core.datatypes.matrix.Matrix(*args, **kwargs)¶

Bases: object

4x4 matrix class.

The following values ​​can be specified in the constructor.

  • Matrix

  • 16-value sequence

  • Row vector specification of 3x3 part by three Vector.

    • Additionally, specify the translate value using the fourth Vector.

    • Transpose the 3x3 part by specifying the transpose=True option.

mirror(mirrorAxis=0, negAxis=True, t=True)¶

Obtain a matrix mirrored along the specified axis.

Parameters:

  • mirrorAxis (int) – Reference axis to mirror. Specify either AXIS_X, AXIS_Y, or AXIS_Z.

  • negAxis (int) – Local axis to orient the opposite of the mirror result so that the determinant (scale) is not reversed. In addition to AXIS_X, AXIS_Y, AXIS_Z, you can specify None, False, and True. If omitted (True), it will be the same as the mirror reference axis. If False, the 3x3 part will not be reversed. None will reverse all axes from the mirror result.

  • t (bool) – Whether the translation value (4th line) is also inverted.

Return type:

Matrix

mirrorIt(mirrorAxis=0, negAxis=True, t=True)¶

Sets the mirrored matrix along the specified axis.

Parameters:

  • mirrorAxis (int) – Reference axis to mirror. Specify either AXIS_X, AXIS_Y, or AXIS_Z.

  • negAxis (int) – Local axis to orient the opposite of the mirror result so that the determinant (scale) is not reversed. In addition to AXIS_X, AXIS_Y, AXIS_Z, you can specify None, False, and True. If omitted (True), it will be the same as the mirror reference axis. If False, the 3x3 part will not be reversed. None will reverse all axes from the mirror result.

  • t (bool) – Whether the translation value (4th line) is also inverted.
