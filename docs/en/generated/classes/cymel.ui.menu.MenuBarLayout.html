class cymel.ui.menu.MenuBarLayout¶

class cymel.ui.menu.MenuBarLayout(*args, **kwargs)¶

Bases: Layout

menuBarLayout wrapper class for mel UI.

setParent can be done with with.

Based on Maya standard commands, you must use methods such as menus instead of children to obtain laid out menus. However, you can get the layout from Menu with Control.parent.
