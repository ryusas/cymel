cymel for Maya¶

An open source Python module that makes Maya scripting as concise and intuitive as Python. It was developed with great influence from pymel, but it allows for lighter and more nimble programming than the original.

C in cymel is C in C++. The core is implemented in C++ and aims to run at high speed. It’s currently a preview version written entirely in Python, but it’s still very lightweight.

The core of the official version will be replaced by a C++ implementation, but the plan is to maintain the Python version and allow it to be switched at any time. The Python version allows you to quickly complete tasks that would require a lot of effort in a C++ implementation, such as when major changes are made when upgrading Maya, and it also guarantees the minimum level of operation even if binaries cannot be built on all platforms.
