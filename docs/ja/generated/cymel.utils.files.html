cymel.utils.files.exportSceneFile(fname, sel=None, all=False, ftype=None, **kwargs)¶

シーンファイルをエクスポートする。

パラメータ:

  • fname (str) -- シーンファイル名。 ディレクトリが存在しない場合は自動的に生成される。

  • sel -- all=False の場合に出力するノードの名前やリスト。 省略すると選択されているものがエクスポートされる。

  • all (bool) -- シーン全体をエクスポートするかどうか。

  • ftype (str) -- ファイルタイプ名。拡張子から推測できる場合は省略可能。

戻り値の型:

str

cymel.utils.files.importSceneFile(fname, namespace=None, prefix=None, **kwargs)¶

シーンファイルをインポートする。

パラメータ:

  • fname (str) -- シーンファイル名。

  • namespace (str) -- インポートしたノードにネームスペースを付加する場合に指定する。 文字列ではない真値を指定すると、ファイル名から自動決定される。

  • prefix (str) -- インポートしたノードにプレフィックスを付加する場合に指定する。 文字列ではない真値を指定すると、ファイル名から自動決定される。

戻り値の型:

str

パラメータ:

  • addRecent (bool) -- Recent Files に追加するかどうか。

  • prefix (str) -- リファレンスしたノードにプレフィックスを付加する場合に指定する。 文字列ではない真値を指定すると、ファイル名から自動決定される。

パラメータ:

  • ftype (str) -- ファイルタイプ名。拡張子から推測できる場合は省略可能。
