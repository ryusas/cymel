cymel.constants¶

cymel全体で使用する主な定数。

よく使うものに厳選してあるので * で import しても良いかも知れない。

Constants:

cymel.constants.AVOID_ZERO_DIV_PRECISION = 1e-13¶

Maya では scale=0 を設定しても matrix 値は 1.e-12 くらいで潰れずに保持するようなので、それを意識した極小値。
