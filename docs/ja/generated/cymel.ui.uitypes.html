cymel.ui.uitypes.inspectUIInheritance()¶

MayaのUI型について調べる。

バージョンアップ時などに実行して調べるためのもの。 そして、この出力を元に _UICMD_DICT と _UITYPE_TO_UICMD の定義を書く。

  • _UICMD_DICT は値が tuple なものについて確定する。

  • _UITYPE_TO_UICMD はそのまま利用。
