class cymel.compat_nodetype.CompatNodeTypeMap¶

class cymel.compat_nodetype.CompatNodeTypeMap(*args, **kwargs)¶

ベースクラス: object

Mayaバージョンによるノードタイプ名変更に伴う新旧バージョン互換性確保のための名前変換器。

唯一のインスタンス compat_nodetype_map を生成済み。

Methods:

correctName(name)

正しい名前に補正する。

futureName(name)

将来の変更後の名前を得る。

Methods Details:

correctName(name)¶

正しい名前に補正する。

getattr ではエラーになる 旧バージョンでは存在しないもの（同名でも意味が変わってしまうものも含まれる） を指定しても、それが存在する新バージョンではエラーにならず、 旧バージョンでは None が返される。

パラメータ:

name (str) -- ノードタイプ名。

戻り値の型:

str or None

futureName(name)¶

将来の変更後の名前を得る。

パラメータ:

name (str) -- ノードタイプ名。

戻り値の型:

str
